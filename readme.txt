1. Informations
  A script which makes bombshops and pay'n'sprays useful.


2. Installation
  a) Unzip 'extended_garages.zip' to your 'Scripts' directory.
  b) Open 'content.xml' and add '<script folder="extended_garages"/>' in the bottom.


3. Settings (server side)
  You can change some basic settings easly in the 'config.ini' file.
